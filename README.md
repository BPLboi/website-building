# Website Builder

Uses a specialized markdown format to create webpages.

## Goals

Defaults that should work well:
- Should have site navigation bar
- Should be able to get most of the default HTML things with markdown, with default style formatting
    - Headers
    - Paragraphs
    - Lists (bullet/enumerated), should have styling options
    - 
- Image embedding

Some nice formatting 

Specialized things worth having:
- Should be able to create a block which uses slightly different styling rules (CSS?)
    - Should be able to specify custom CSS
    - Should be able to specify an alternate style set for some elements?
- Support LaTeX
- Minipages, tables, etc.
- Should be able to write directly in HTML if necessary
- Should be able to embed canvases/javascript windows
- Footnotes/comments on the side
- Dark mode? (native?)
